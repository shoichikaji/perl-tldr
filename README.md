# tldr [![Build Status](https://travis-ci.com/skaji/perl-tldr.svg?branch=master)](https://travis-ci.com/skaji/perl-tldr)

> a perl client for [tldr](http://tldr-pages.github.io/)

- Install

`cpanm -nq App::tldr`

- Show `tar` tldr

`tldr tar`

- OS specific (linux, osx or sunos)

`tldr -o osx brew`
